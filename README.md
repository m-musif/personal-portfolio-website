# Muhammad Musif — Personal Portfolio

Professional portfolio for Muhammad Musif: AI engineer, machine learning developer, and full-stack AI builder.

**Live site:** https://m-musif.github.io/personal-portfolio-website

## Stack

- React + Vite
- Tailwind CSS v4
- GitHub Pages (GitHub Actions deploy)

## Local development

```bash
cd frontend
npm install
npm run dev
```

## Production build

```bash
cd frontend
npm install
npm run build
```

## Deployment (GitHub Pages)

1. Push changes to the `main` branch.
2. In **Settings → Pages**, set **Source** to **GitHub Actions**.
3. The workflow in `.github/workflows/deploy.yml` builds `frontend/` and publishes to Pages.
